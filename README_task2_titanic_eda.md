# 🚢 Titanic EDA — Survival Analysis

Exploratory Data Analysis on the famous Titanic dataset to uncover survival patterns based on gender, age, and fare.

## 📊 Project Overview

This project performs in-depth EDA on the Titanic passenger dataset. It explores how factors like gender, age, and ticket fare influenced survival rates during the disaster.

## 📸 Sample Output

> - Bar chart: Survival count by Gender (Male vs Female)
> - Scatter plot: Age vs Fare colored by Survival status

## 🛠️ Tech Stack

- **Python**
- **Pandas** — data cleaning and analysis
- **Seaborn** — statistical visualizations
- **Matplotlib** — custom plots

## 📁 Dataset

- Titanic Dataset (available on Kaggle)
- Features: PassengerId, Survived, Pclass, Name, Sex, Age, Fare, Embarked


## 📌 Key Insights

- Females had a significantly higher survival rate than males
- Passengers who paid higher fares had better survival chances
- Younger passengers had slightly higher survival rates
- Most passengers were in the age range of 20–40 years
